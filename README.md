# 💰 Finance Tracker (Local-First)

Una aplicación web premium, local-first y centrada en la privacidad para gestionar tus finanzas personales.

![Dashboard Preview](https://via.placeholder.com/800x450?text=Finance+Tracker+Dashboard)

## ✨ Características

- **Visualización Avanzada**: Gráficos interactivos de evolución histórica y desgloses de gastos por categoría (Chart.js).
- **Control de Movimientos**: Registro de ingresos y gastos con categorización automática.
- **Plantillas de Gastos Fijos**: Define tus gastos recurrentes y cárgalos al mes actual con un solo clic.
- **Reconocimiento de Marcas**: Etiquetas automáticas con colores corporativos e iconos para marcas habituales (Mercadona, Amazon, Netflix, etc.).
- **Privacidad Total**: Los datos nunca salen de tu dispositivo. Se guardan en el `localStorage` de tu navegador.
- **Añadir en Bloque**: Herramienta de importación masiva para múltiples movimientos.

## 🚀 Instalación y Uso

Este es un proyecto estático. No necesitas instalar bases de datos complicadas.

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/finanzas.git
   ```
2. Ejecuta el archivo `start.bat` (en Windows) o simplemente abre el archivo `index.html` en tu navegador favorito.
3. Para una experiencia más completa con un servidor local, puedes usar:
   ```bash
   npx serve .
   ```

## 🛠️ Tecnologías

- **HTML5** & **CSS3** (Diseño Premium con Inter Font)
- **Vanilla JavaScript** (ES6+)
- **Chart.js** (Visualización de Datos)
- **FontAwesome** (Iconografía)

## 📋 Normativa y GitHub Pages

Si deseas publicar este proyecto en **GitHub Pages**:

1. Ve a **Settings > Pages** en tu repositorio de GitHub.
2. En **Build and deployment**, selecciona la rama `main` y la carpeta `/ (root)`.
3. **IMPORTANTE**: No rellenes el campo "Custom domain" con la palabra "finanzas" a menos que tengas un dominio real comprado. GitHub te asignará automáticamente una URL tipo `https://tu-usuario.github.io/finanzas/`.
